### 기본형과 참조형의 종류 및 차이점     
---
          
#### 기본형(Primitive Type)
 - Number
 - String
 - Boolean
 - null
 - undefined
 - (es6에선 symbol 추가)
           
#### 참조형(Reference Type)
 - Object
   - Array
   - Function
   - RegExp
   - (es6에선 map, set 등 추가)
              
```
기본형은 값을 그대로 할당하고, 참조형은 값이 저장된 주소값을 할당(참조)
```
